> 整理自2024-2025年最新论文、专利和工程实践，涵盖变电站场景下多传感器融合SLAM、自主导航、路径规划等领域。

## 一、多传感器融合SLAM技术

### 1.1 融合架构分类

| 架构 | 描述 | 优点 | 缺点 | 代表方法 |
|------|------|------|------|----------|
| **松耦合** | 各传感器独立计算位姿，最后加权/投票融合 | 模块化、易调试、计算量小 | 信息利用不充分，精度受限 | 早期多传感器方案 |
| **紧耦合** | 传感器原始数据层面联合优化 | 精度高、鲁棒性强 | 实现复杂、计算量大 | FAST-LIO2, LIO-SAM, R3LIVE |
| **因子图优化** | 将所有约束建模为因子图中的因子，全局优化 | 灵活、可扩展、精度最高 | 延迟较高 | GTSAM, LIO-SAM, LIRO-SLAM |

### 1.2 传感器组合方案对比

| 组合 | 适用场景 | 定位精度 | 抗EMI能力 | 成本 |
|------|---------|---------|-----------|------|
| LiDAR + IMU | 室内外通用 | ±2~5cm | ★★★★ | 中 |
| Visual + IMU | 室内、光照好 | ±3~10cm | ★★ | 低 |
| LiDAR + IMU + Visual | 复杂光照/退化环境 | ±1~3cm | ★★★★ | 高 |
| LiDAR + IMU + GNSS/RTK | 室外变电站 | ±1~5cm | ★★★ | 高 |
| LiDAR + IMU + Visual + 超声 | 高压室内 | ±1~3cm | ★★★★★ | 很高 |

## 二、2024-2025年代表性工作详析

### 2.1 LIRO-SLAM（IEEE 2024）

> **LiDAR-Inertial SLAM for UAV Navigation in UHV Substations**

- **发表**：IEEE国际会议，杭州，2024年10月
- **核心技术**：LiDAR里程计 + IMU预积分紧耦合，基于图优化框架
- **关键技巧**：关键帧选择 + 运动约束 + IMU偏置在线校正
- **性能**：在特高压变电站实际测试，优于现有SOTA SLAM算法
- **适用性**：特高压变电站/换流站的无人机/地面机器人巡检
- **论文链接**：[IEEE Xplore](https://ieeexplore.ieee.org/document/10824643)

### 2.2 VIORB-SLAM2（Energy Informatics, 2024）

> **Improved SLAM for Substation Inspection Robot Based on IMU and Visual Fusion**

- **发表**：Energy Informatics, Vol.7, Article 86, 2024
- **解决的问题**：ORB-SLAM2在特征稀疏/低光照环境下的跟踪失败和尺度不确定性
- **方案**：
  - IMU数据恢复真实尺度信息
  - 联合视觉重投影误差 + IMU预积分误差构建代价函数
  - RANSAC剔除误匹配 + NCC归一化互相关做关键帧约束
  - ICP生成稠密点云地图
- **链接**：[Springer Open](https://link.springer.com/article/10.1186/s42162-024-00390-8)

### 2.3 CN117968666A 专利（2024）

> **基于融合SLAM的变电站巡检机器人定位与导航方法**

- **完整流程（六大步骤）**：
  1. **系统初始化**：检测激光雷达、双目摄像头状态
  2. **信息融合与建图**：LiDAR点云 + 双目视觉 → 卡尔曼滤波紧耦合 → 回环检测保证一致性
  3. **全局路径规划**：A*算法智能规划巡检路径
  4. **局部避障**：D*动态搜索算法实时避障
  5. **地图更新**：巡检过程中持续更新环境地图
  6. **任务收尾**：生成巡检报告，优化算法参数

- **运行模式**：全遥控 / 半自动 / 全自动（三级可切换）
- **通信**：5G远程传输
- **链接**：[Google Patents](https://patents.google.com/patent/CN117968666A/zh)

## 三、核心SLAM框架对比

### 3.1 主流开源框架一览

| 框架 | 传感器 | 融合方式 | 后端 | 适用场景 | 特点 |
|------|--------|---------|------|---------|------|
| **FAST-LIO2** | LiDAR+IMU | 紧耦合IEKF | 滤波 | 快速运动、退化环境 | 速度快、鲁棒性好 |
| **LIO-SAM** | LiDAR+IMU+GPS | 紧耦合因子图 | GTSAM优化 | 室外大场景 | 回环检测、精度高 |
| **FAST-LIO-SAM** | LiDAR+IMU | LIO2前端+SAM后端 | GTSAM | 室内外通用 | 兼具速度与精度 |
| **R3LIVE** | LiDAR+IMU+Visual | 紧耦合 | 滤波+优化 | 室内外通用 | 可彩色建图 |
| **FAST-LIVO** | LiDAR+IMU+Visual | 紧耦合 | 滤波 | 快速运动 | R3LIVE的加速版 |
| **ORB-SLAM3** | Visual(+IMU) | 松/紧耦合可选 | 优化 | 室内、纹理丰富 | 最成熟的视觉SLAM |
| **VINS-Mono** | Visual+IMU | 紧耦合 | 滑窗优化 | 无人机/手持 | 经典的VIO方案 |
| **Cartographer** | LiDAR(+IMU) | 松耦合 | 图优化 | 室内2D建图 | Google出品，稳定 |

### 3.2 针对变电站场景的推荐路线

```
入门：FAST-LIO2（简单、鲁棒、教程多）
  ↓
进阶：LIO-SAM 或 FAST-LIO-SAM（加回环检测，应对重复结构）
  ↓
完善：R3LIVE / FAST-LIVO（加入视觉，解决金属反射问题）
  ↓
定制：在上面基础上加入变电站特定约束和抗干扰模块
```

## 四、自主导航与路径规划

### 4.1 导航栈架构（ROS2 Nav2）

```
┌─────────────────────────────────────────────┐
│                 Nav2 导航栈                    │
├─────────────────────────────────────────────┤
│  planner_server    → 全局路径 (A*/Dijkstra)   │
│  controller_server → 局部控制 (DWA/TEB/MPC)   │
│  recoveries_server → 恢复行为 (旋转/后退)      │
│  bt_navigator      → 行为树 (任务编排)        │
│  costmap_2d        → 全局/局部代价地图         │
│  waypoint_follower → 多航点巡检               │
└─────────────────────────────────────────────┘
```

### 4.2 路径规划算法对比

| 算法 | 类型 | 适用 | 特点 |
|------|------|------|------|
| **A*** | 全局 | 静态已知地图 | 经典启发式，最优解 |
| **Dijkstra** | 全局 | 静态已知地图 | 无启发式但保证最短 |
| **D*** | 全局动态 | 环境部分已知 | 增量重规划，效率高 |
| **DWA** | 局部 | 动态避障 | 速度空间采样，实时性好 |
| **TEB** | 局部 | 复杂约束 | 时空轨迹优化 |
| **MPC** | 局部 | 精准控制 | 预测控制，约束处理好 |
| **RRT/RRT*** | 全局 | 高维空间 | 概率采样，不依赖栅格 |
| **强化学习** | 端到端 | 复杂动态环境 | 数据驱动，泛化性待验证 |

### 4.3 巡检路径规划的特殊需求

变电站巡检不同于一般导航，有其特殊需求：

1. **全覆盖路径规划（CCPP）**：需要保证巡检点位100%覆盖
2. **设备优先级**：高温/重要设备优先巡检（动态AHP权重）
3. **约束路径**：高压区域有安全距离和禁入区域（Keepout Zone）
4. **重复巡检**：同一路径反复执行，路径记忆和平滑优化
5. **多机协同**：划分巡检区域，避免碰撞和重复

## 五、高压室特殊问题与对策

### 5.1 电磁干扰（EMI）问题

| 影响 | 机理 | 对策 |
|------|------|------|
| LiDAR点云噪点增多 | 电磁场干扰激光接收器 | IMU紧耦合降低对单帧点云依赖 |
| 电子罗盘偏移 | 磁场干扰地磁传感器 | 用光纤陀螺或视觉航向修正 |
| 通信丢包 | 5G/WiFi信号被屏蔽 | 本地计算为主，5G为辅助 |
| 电机编码器跳变 | 电磁脉冲干扰 | 光电编码器 + 软件中值滤波 |

### 5.2 金属结构多路径反射

激光雷达在金属柜体/管道表面会产生镜面反射：

- **问题**：LiDAR"看到"不存在的障碍物（镜像点云）
- **检测方法**：基于反射强度异常 + 多帧不一致性
- **解决思路**：
  - 视觉SLAM做第二意见校验（相机不受镜面反射影响）
  - 多点云帧投票去除不稳定特征
  - 深度学习滤除镜像噪声

### 5.3 GPS完全失效

高压室为全金属遮蔽结构，GPS信号完全不可用：

- **定位方案**：纯SLAM定位（不依赖任何外部信号）
- **初始位姿**：设定固定充电桩位置为世界坐标原点
- **累积漂移**：回环检测 + 充电桩回充时自动归零
- **辅助定位**：近超声定位/UWB（可选补充）
