   - 期刊：广东电力 / OPAJ
   - 链接：[opaj.napstic.cn](https://opaj.napstic.cn/periodicalArticle/0120250601131547)

   - 期刊：自动化学报 (Acta Automatica Sinica)
   - 链接：[aas.net.cn](http://www.aas.net.cn/cn/article/pdf/preview/10.16383/j.aas.c230809.pdf)

### SLAM方法类

3. **LIRO-SLAM: Tightly-Coupled LiDAR-Inertial SLAM for UAV Navigation in UHV Substations**（IEEE, 2024）
   - 链接：[IEEE Xplore](https://ieeexplore.ieee.org/document/10824643)

4. **Improved SLAM Algorithm for Substation Inspection Robot Based on IMU and Visual Fusion**（2024）
   - 期刊：Energy Informatics, Vol.7, Art.86
   - 链接：[Springer](https://link.springer.com/article/10.1186/s42162-024-00390-8)

5. **Autonomous Drone Inspection of Substations Under Strong Electromagnetic Interference**（2024）
   - 会议：5th ICBAIE 2024
   - 链接：[Semantic Scholar](https://www.semanticscholar.org/paper/Autonomous-Drone-Inspection-of-Substations-Under-Zou-Zhu/c8300ec3bd74be99b9bcc6a2b28a1792ad084810)

### 具身智能方向

6. **Potential Implementation of Embodied Intelligence Technology in the Power Grid: A Review**（2025）
   - 链接：[scholar.xjtlu.edu.cn](https://scholar.xjtlu.edu.cn/en/publications/potential-implementation-of-embodied-intelligence-technology-in-t/)

7. **Embodied Intelligence-Driven Framework for UAV Power Inspection Systems**（2025）
   - 链接：[Springer](https://rd.springer.com/chapter/10.1007/978-981-96-9964-3_25)

### 路径规划类

8. **Path Planning of Substation Inspection Robot Based on High-Precision Positioning and Navigation**（2024）
   - 期刊：Int. J. of Low-Carbon Technologies
   - 链接：[Oxford Academic](https://academic.oup.com/ijlct/article/doi/10.1093/ijlct/ctae125/7719005)

### 专利

9. **CN117968666A — 基于融合SLAM的变电站巡检机器人定位与导航方法**（2024）

## 🎓 建议学习顺序

```
第1步：ROS2基础 → 古月居ROS2入门21讲
第2步：SLAM理论 → 《视觉SLAM十四讲》前8章
第3步：仿真跑通 → TurtleBot3 + Gazebo + SLAM Toolbox
第4步：导航上手 → Nav2 + 自主导航demo
第5步：激光SLAM → FAST-LIO2 跑自己的数据
第6步：传感器融合 → robot_localization EKF
第7步：深入优化 → LIO-SAM / R3LIVE
第8步：变电站适配 → 读论文 → 仿变电站场景 → 迭代
```
